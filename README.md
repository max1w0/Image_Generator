# Image Generator

A **.NET Core 3.1** Windows Forms application that generates composite images by allowing users to drag and drop blocks onto a grid. The output image is assembled from fragments retrieved from a connected SQL database.

---

## Overview

The **Image Generator** is a Windows Forms application that allows users to create custom images by arranging draggable blocks on a 4x4 grid. Each block represents a specific image fragment, and the final image is generated by combining these fragments with a randomly selected background.

The application connects to a **SQL Server database** to retrieve:
- **Background images** (stored as binary data)
- **Element images** (stored as binary data)

---

## Features

| Feature | Description |
| :--- | :--- |
| **Drag-and-Drop Interface** | Drag blocks from the storage panel onto the grid |
| **Block Types** | Three block sizes: 1x1, 1x2, and 1x3 |
| **Image Generation** | Combines selected blocks with a random background |
| **Database Integration** | Retrieves images from SQL Server |
| **Custom Form Styling** | Borderless forms with custom title bars |
| **Real-time Feedback** | Status updates in the output panel |
| **Sound Feedback** | Play sound on generation errors |
